# crane logo explorations

Four directions inspired by the **crane bird** (catch a thought mid-flight, set it down gently). Brand colors: **electric blue** `#2400FF`, **cream** `#F7E6C8`.

| Folder | Concept | Best for |
|--------|---------|----------|
| `direction-a-neck-drop/` | Side-profile neck + beak dot (the dot = a captured drop) | App icon, strongest bird + product read |
| `direction-b-cr-hybrid/` | Evolves existing `cr.` wordmark; period = drop at beak | Low-risk rebrand from current icon |
| `direction-c-origami/` | Angular folded paper crane | Landing, marketing |
| `direction-d-wingspan/` | Single flight arc (abstract lift) | Menu bar, minimal surfaces |

Each folder contains:

- `app-512.svg` — squircle app icon (512×512)
- `menubar-16.svg` — simplified mark for 16×16 menu bar

Preview in Finder, Figma, or any browser (`open icons/preview.html`). SVGs use ASCII-only labels (no special punctuation) so macOS Preview and Xcode open them cleanly.

**Selected for production:** `direction-d-wingspan/` (wired into `crane/AppIcon.svg`, `MenuBarIcon.imageset`, `landing/assets/app-icon.svg`, and `AppIcon.appiconset` PNGs).
